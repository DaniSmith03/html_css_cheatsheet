HTML Elements Reference

HTML Element

Decription

Example
<head> The element is a container for metadata (data about data) and is placed between the tag and the tag. <head>
<title>Title of the document </title>
</head>
<body> This element contains all the contents of an HTML document, such as headings, paragraphs, images, hyperlinks, tables, lists, etc.. <body>
<h1>Paragraph header </h1>
<p>Some text content </p>
</body>
<div> this element is used as a container for HTML elements - which is then styled with CSS or manipulated with JavaScript. This tag defines a division or a section in an HTML document. <body>
<div>
<h1>Paragraph header </h1>
<p>Some text content </p>
</div>
</body>
<h1>-<h6> These elements define headings from most important to least important. <h1>This is heading 1</h1>
<h6>This is heading 6</h6>
<p> This element defines a paragraph, used for text <p>Some example paragraph text.</p>
<p>each p tag will generate a new paragraph with spacing.</p>

CSS Properties Reference

CSS Properties

Decription

Example
color The color property specifies the foreground color of text body {color: #92a8d1;}
p {color: #FFFFFF;}
background-color This property sets the background color of an element (the space occupied within it's margins) body {background-color: #92a8d1;}
h1 {background-color: #92a8d1;}
font-family This property sets the font to be used for an element or multiple elements * { font-family: Arial, Helvetica, sans-serif; }
border Sets the style (width, color, type) for a specific element. header {border: 5px dotted black;}
text-align aligns the text for HTML elements. Default valuse is left, this can be changed to right or center. h1 { text-align: center;}
